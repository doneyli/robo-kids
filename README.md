# robo-kids

Teaching robotics to kids (ages 4-8) with Reachy Mini.

## What is this?

A family project exploring different approaches to introducing two kids (ages 4 and 8) to robotics concepts using a [Reachy Mini](https://huggingface.co/docs/reachy_mini) desktop humanoid robot.

## Approaches Being Explored

| # | Approach | Description |
|---|----------|-------------|
| # | App | Folder | Description |
|---|-----|--------|-------------|
| 1 | **RoboQuest** | `robo-quest/` | Gamified Next.js web app with age-differentiated tracks |
| 2 | **Robo Curriculum** | `robo-curriculum/` | 12-week static site with lesson plans for ages 4 & 8 |
| 3 | **Kids Commander** | `kids-commander/` | Kid-friendly Reachy Mini control interface (Python/Streamlit) |

## Constraints

- **1-2 hours/week** of dedicated time
- **Hardware available** — Reachy Mini physically set up; all apps include simulator fallback
- **Age-appropriate** — separate tracks for 4yo (play-based) and 8yo (structured learning)

## Getting Started

Each approach lives on its own branch. See the individual PRs for setup instructions.
